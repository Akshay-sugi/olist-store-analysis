# 🛒 Olist E-Commerce Analysis

## 📌 Project Overview
This project analyzes Brazil’s Olist E-Commerce dataset to uncover sales trends, customer behavior, delivery performance, and payment insights using SQL, Power BI, Tableau, and Excel.

The analysis focuses on identifying business growth opportunities, improving delivery performance, and understanding customer purchasing patterns through interactive dashboards and KPI reporting.

---

## 🛠 Tools & Technologies
- MySQL
- Power BI
- Tableau
- Excel
- SQL
- Data Visualization

---

## 📊 Key Insights
✅ Weekday sales generated higher revenue than weekends  
✅ Faster deliveries received better customer reviews  
✅ São Paulo contributed the highest number of orders  
✅ Credit cards were the most preferred payment method  
✅ Bed, Bath & Table category had the highest sales
